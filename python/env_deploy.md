# 安装pyenv
- 官方安装方式
```shell script
curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash
```

- 国内安装方式

由于从github上clone很慢, 使用[ghproxy](https://ghproxy.com/)加速, 修改`pyenv-installer`后, 执行
```shell script
bash pyenv-installer
```
